# HealthSync

个人健康管理工具 - 通过华为 Health Kit 同步手环和体脂秤数据

## 文档

- [隐私政策](https://github.com/yan9708172-prog/healthsync/blob/main/%E9%9A%90%E7%A7%81%E5%8D%8F%E8%AE%AE)
- [用户协议](https://github.com/yan9708172-prog/healthsync/blob/main/%E7%94%A8%E6%88%B7%E5%8D%8F%E8%AE%AE)

## 功能

- 同步华为手环数据：步数、消耗热量、心率、睡眠
- 同步华为体脂秤数据：体重、体脂率、BMI、骨骼肌量等
- 健康数据趋势分析
- 日/周/月健康报告生成
